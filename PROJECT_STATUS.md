## 📞 Quick Start

### Demo (No Setup Required)
```bash
# Just open in browser
open demo/index.html
```

### Full Application
```bash
# 1. Setup backend
cd backend
cp .env.example .env
# Add your AWS credentials and YouTube API key
bun install
bun run migrate
bun run dev

# 2. Setup frontend
cd frontend
npm install
npm run dev

# 3. Visit http://localhost:3000
```
